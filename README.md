①課題名
　Moodlog
②課題内容（どんな作品か）
　気分を5段階のアイコンとメモで記録できるアプリです。
③アプリのデプロイURL
　https://mitsuko117.github.io/Moodlog/
④アプリのログイン用IDまたはPassword（ある場合）
　なし
⑤工夫した点・こだわった点
　気分アイコンを選択したときにサイズが大きくなるため、メモ欄と重ならないよう配置を調整しました。
　入力エリアも適度にスペースを取り、文字を打ち込んでいる際に不細工な見た目にならないようにしました。
⑥難しかった点・次回トライしたいこと（又は機能）
　本来は前回のカレンダーアプリに追加する形で実装したかったのですが、うまく動作せず…
　今回は気分とメモの記録に機能を絞りました。
⑦フリー項目（感想、シェアしたいこと等なんでも）
　　以前作ったコードを再利用したので、スムーズに進められましたし、理解も前より深まったと感じています。
　卒業制作を見据えて作っているので、パーツとして再利用しやすい構成にしています。
　今後も卒業制作に使える部品を少しずつ作っていくつもりです。
